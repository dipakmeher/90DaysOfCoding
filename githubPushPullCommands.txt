ChangeDirectory: cd 
Create folder: mkdir FolderName
Create file: type FileName
Clond Directory: git clone https://github.com/dipakmeher/90DaysOfCoding.git
Optional: git config user.name "dipakmeher"
Optional: git config user.email "dipakmeher54@gmail.com"
Optional: git remote add origin <Link to GitHub Repo>     //maps the remote repo link to local git repo

Optional: git remote -v                                  //this is to verify the link to the remote repo 

Check branch: git branch
Create new branch: git branch Branch_Name
Create Branch and Switch to it: git checkout -b Branch_Name
Go to new branch: git checkout Branch_Name
Check Status: git status
Add Directory in Push: git add .
Check Status: git status
Commit to Repo: git commit -m "Put Message"
Push the Branch to Git: git push --set-upstream origin Day15

Change to Main: git checkout main   
Merge branches: git merge Branch_Name
List the last 5 commits: git log --oneline
====
Push to Main Branch: git push -u origin main
